# Тест на биохимический тип личности (тест Бравермана)

Лёгкий одностраничный сайт: 200 вопросов (Да/Нет), один сплошной тест (~20 минут),
график по 4 нейромедиаторам в абсолютных баллах, определение доминирующего типа
и активной доли мозга, автосохранение результатов в Google-таблицу.

## Что внутри

| Файл | Назначение |
|------|-----------|
| `index.html` | Разметка трёх экранов: вступление → вопросы → результат |
| `styles.css` | Светлый минималистичный дизайн, адаптив под телефон |
| `content-ru.js` | Русская версия: интерфейс, описания типов и 200 вопросов |
| `content-kk.js` | Казахская версия: интерфейс, описания типов и 200 вопросов |
| `data.js` | Ядро: цвета нейромедиаторов, порядок осей, реестр языков |
| `app.js` | Логика: язык, навигация, подсчёт, график, отправка в таблицу |
| `apps-script.gs` | Код Google Apps Script для приёма результатов |

## Языки

Тест доступен на **русском** и **казахском**. Переключатель «Рус / Қаз» — в правом
верхнем углу стартового экрана; выбор запоминается в браузере. Казахские
формулировки написаны простым разговорным языком с естественным для казахского
порядком слов (глагол в конце), без книжных оборотов.

Вопросы в обоих языках идут в одинаковом порядке, поэтому язык можно переключить
даже посреди теста — ответы не потеряются. В таблицу пишется колонка «Язык».

## Нейромедиаторы и доли мозга

| Нейромедиатор | Доля мозга | Тип |
|---------------|-----------|-----|
| Дофамин | Лобная | Энергичный лидер |
| Ацетилхолин | Теменная | Творческая натура |
| ГАМК | Височная | Надёжная опора |
| Серотонин | Затылочная | Живущий настоящим |

Каждый блок — 50 вопросов. Да = 1 балл, Нет = 0. Результат блока — это
**абсолютный балл** (число «Да», от 0 до 50). Доминирующий нейромедиатор —
с наибольшим баллом, его доля мозга считается активной. Вопросы перемешаны,
поэтому проходящий не видит деления на блоки — для него это единый тест.

## Запуск локально

```bash
cd "Abdrashit AI"
python3 -m http.server 8000
```
Откройте http://localhost:8000

(Можно просто открыть `index.html` двойным кликом — тест работает,
но для корректной отправки в таблицу лучше через http-сервер или хостинг.)

## Подключение Google-таблицы (сбор результатов)

1. Создайте новую **Google-таблицу**.
2. В ней: **Расширения → Apps Script**.
3. Удалите шаблон, вставьте содержимое `apps-script.gs`, сохраните.
4. **Развернуть → Новое развёртывание → Веб-приложение**:
   - «Выполнять от имени»: **Я**
   - «У кого есть доступ»: **Все**
5. Скопируйте выданный **URL веб-приложения**.
6. Откройте `app.js` и вставьте этот URL в строку:
   ```js
   const SHEET_ENDPOINT = 'PASTE_YOUR_APPS_SCRIPT_URL_HERE';
   ```
7. Готово: после каждого прохождения в таблицу добавляется строка
   (дата, ФИО, 4 балла (из 50), доминирующий тип, активная доля мозга).

Пока `SHEET_ENDPOINT` не заполнен, тест работает полностью — просто
результаты не сохраняются на сервере.

## Деплой (чтобы делиться ссылкой)

Самый быстрый способ — перетащить папку на **[Netlify Drop](https://app.netlify.com/drop)**.
Также подойдут **GitHub Pages** или **Vercel** (это статические файлы, без сборки).

## Как редактировать вопросы

Русские вопросы — в `content-ru.js`, казахские — в `content-kk.js` (по 50 на блок).
Можно менять формулировки — число вопросов в блоке определяется автоматически
(переплетение блоков и подсчёт баллов подстраиваются сами).

⚠️ Важно: порядок вопросов в обоих файлах должен совпадать — вопрос №7 в
`content-ru.js` и №7 в `content-kk.js` должны быть одним и тем же утверждением.
